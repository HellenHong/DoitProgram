ratings.txt : 모델 학습을 위한 예제 파일. 실사용할 파일은 저희가 만들어야 합니다!


저희는 딥러닝 챗봇을 구현하여, 다음과 같은 편의 목적을 이루고자 합니다.
참고 자료 : 처음 배우는 딥러닝 챗봇 (도서)

1. 빠르게 기능 찾기
2. 추천 강의계획서 찾기
3. 수강신청 관련 일정 알아보기
4. 교수님이 강의하시는 강의 알아보기
5. 어느 강의가 어느 학기에 열리는지 알아보기
6. 단순 잡담
등등... 다양한 방향성이 가능할 것으로 생각됩니다.

챗봇 대화 유형 예상
- 일상적인 안부 및 잡담 (안녕 / 좋은 아침 / 뭐해 / 이름이 뭐야 넌 누구야 / ...)
- 기능 찾기 및 동작 설명 -> 바로가기 연결? (일정 추가 어디서 해 / 강의 추가 어디서 해 / 일정 수정은 어떻게 해...)
- 일정 알려주기 -> 이번 달 일정을 알려줘 / 가까운 일정을 알려줘
- 학교 홈페이지 연결 -> 장학금 관련해서 새로운 공지가 있어? 컴퓨터공학과 공지가 있어? 어디 학과 교수님 목록 알고싶어 등등...?

-------

사용할 모델은 Word2Vec 모델입니다.
Word2Vec 모델은 CBOW와 skip-gram 두 가지가 있습니다. 각각 주변으로부터 특정 단어 추측 / 특정 단어로부터 주변 단어를 추측하는 모델입니다.
-> Gensim 패키지를 이용합니다.

모델 학습을 위해서는 다양한 말뭉치가 필요합니다. 학습이 완료된 모델은 파일로 저장하여 사용합니다.

문제는 학습을 위한 말뭉치 파일을 제작하는 것입니다. 실습을 위해 참고한 책에서 제공하는 파일을 사용하였습니다. (https://github.com/e9t/nsmc)
